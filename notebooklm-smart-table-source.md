## Short Product Summary

Smart Table is a digital service platform for restaurants and cafes. It allows every table in a venue to have its own QR code. When a guest scans the QR code, they can open the restaurant menu, browse available items, and request service from their phone without downloading an app.

For the restaurant team, Smart Table provides a live dashboard where owners, managers, and waiters can see table requests, respond faster, manage tables, organize staff accounts, update the menu, and monitor service activity.

In simple words: Smart Table turns each table into a direct communication point between the guest and the staff.

## How Smart Table Works

Smart Table works in three simple steps.

Step one: The owner or manager creates the restaurant tables inside the dashboard.

Each table receives a unique QR code. For example: Table 1, Table 2, Terrace 4, VIP Room 1, or any name the restaurant prefers.

Step two: The restaurant prints and places the QR codes on the tables.

Guests scan the QR code using their phone camera. The guest does not need to download an app, create an account, or log in.

Step three: The guest uses the table page.

From the table page, the guest can browse the menu and send a waiter request. Staff see the request in their dashboard and can respond based on the exact table number.

## Core Features

### QR Code Table Access

Each table gets its own QR code. When scanned, the QR code opens the correct table session. This means the restaurant knows exactly which table is asking for service.

### No Customer App Required

Guests use their normal phone browser. This removes friction because customers do not want to download a new app just to ask for service or view a menu.

### Digital Menu

The restaurant can create a digital menu with products, categories, prices, and images. Menu changes can be made from the dashboard without reprinting the whole menu.

### Live Waiter Requests

Guests can request staff directly from the table page. The request appears in the staff dashboard so the team can respond faster.

### Owner Dashboard

The owner dashboard gives the business a clear view of table activity and service requests. It helps managers understand what is happening during service.

### Waiter Accounts

Restaurant owners can create waiter accounts so staff can access the service dashboard. This keeps the system organized and separates owner permissions from staff permissions.

### Table Management

Owners can create, update, block, unblock, and manage tables. QR codes can be generated for each table.

### Product and Category Management

Owners can organize the menu into categories such as drinks, coffee, desserts, pizza, sandwiches, or specials. Products can include names, descriptions, prices, and images.

### Customer Reviews

Smart Table can collect customer reviews from table sessions, giving the restaurant more feedback about the guest experience.

### Business Settings

The restaurant can manage its business profile and settings so the customer-facing experience matches the venue.

### Subscription Plans

Smart Table is offered as a SaaS subscription. Restaurant owners can start with a free trial and then choose a monthly or annual plan based on their needs.
